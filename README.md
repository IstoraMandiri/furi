# Furi

### CLI tool to add anki-style furigana to Japanese text, and get word definitions in english

To Install `furi` globally, clone this Repo and run `npm i -g .`

Example:

```bash
$ furi さすがの私でさえ、グーグル検索はできます。
さすがの私でさえ、グーグル検索はできます。
---

さすがの 私[わたし]でさえ、グーグル 検索[けんさく]はできます。

Even I, as expected, can do a Google search.

~ さすがの as one would expect; just as you'd expect (from); just like (someone); (adv) (uk) still; all the same; after all; (adv) (uk) even ... (e.g. "even a genius")
~ 私[あたし(P); あたくし; あたい; あて] (arch) (ksb:) I; me; (pn)  (1) (arch) (ksb:) I; me; (pn) (2)  (arch) (ksb:) I; me; (pn) (わい only) (used toward people of equal or lower status) you;  【わたい】 ; (pn) (arch) (ktb:) I; me;  【わたくし】 ; (pn) (1) (usu. more formal than わたし) (See 私・わたし) I; me; (adj-no,n) (2) (ant: 公・おおやけ・2) personal (affairs, etc.); private; (n) selfishness; partiality; (n) secrecy; confidentiality;  【わっち】 ; (pn) (used esp. by people of low status) I; me; ED Name(s):  【きさい】 (s) Kis
~ でさえ even (e.g. even a child); (exp,prt) as long as (it is, you are, etc.); (if) only; just
~ グーグル[グーグル] グーグル : Ｇｏｏｇｌｅ 【グーグル】 	(n) Google; ED; Name(s):  ; (u) Google  
~ 検索[けんさく] 検索 【けんさく】 	(n,vs,vt) looking up (e.g. a word in a dictionary); retrieval (e.g. data); searching for; referring to; (P); ED 
~ できます できます 	(aux-v) is able to do; KD 

---
```
